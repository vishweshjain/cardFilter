# 🃏 CARD FILTER

A simple and interactive mini project that allows users to search and filter cards by name in real-time. Built using **HTML, CSS, and JavaScript**, this project demonstrates DOM manipulation and dynamic UI updates.

- ⚡ Instant filtering of cards

- 📱 Responsive design

## 💡 How It Works

1. Cards with names are displayed on the screen  
2. User enters a name in the search bar  
3. JavaScript listens to the input event  
4. Cards are filtered based on matching text  
5. Matching cards are shown, others are hidden
